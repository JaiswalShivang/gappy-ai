# ── Lemma SDK ─────────────────────────────────────────────────────────────────
# The Lemma runtime SDK — provides @fn, inject, Secret decorators,
# pod.files, pod.db injection interfaces, and workflow primitives.
lemma>=1.0.0

# ── Groq (transcription + LLM) ────────────────────────────────────────────────
# Used by: transcribe_audio.fn.py (Whisper transcription)
# The meeting_analyst and email_drafter agents call Groq via Lemma's
# model adapter — NOT via this SDK directly. This is for the Whisper HTTP API.

# ── OpenAI-compatible client (optional fallback) ──────────────────────────────
# Used by: transcribe_audio.fn.py (Groq is OpenAI-API compatible)

# Used by: generate_embedding.fn.py

# sentence-transformers requires torch. CPU-only build keeps size manageable.
# On GPU hosts, replace with the CUDA build.

# ── MongoDB (external vector store — Decision 2) ──────────────────────────────
# Used by: vector_search.fn.py

# (Lemma-managed) — but vector_search calls Atlas externally via Motor.

# ── FFmpeg Python bindings (audio clip cutting) ───────────────────────────────
# Used by: generate_audio_proof.fn.py
